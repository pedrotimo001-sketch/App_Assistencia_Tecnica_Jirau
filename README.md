# App Assistência Técnica Jirau V2 Premium

## Instalação

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Primeiro acesso

- Usuário: `admin`
- Senha: `admin123`

## Principais recursos

- Dashboard executivo e SLA
- Cadastro, edição, pesquisa e filtros
- Agenda de visitas
- Linha do tempo por assistência
- Upload de fotos, laudos e assinaturas
- Ordem de Serviço em PDF
- Usuários e perfis
- Importação e exportação em Excel
- Backup do banco SQLite

A planilha original é importada automaticamente na primeira execução.
